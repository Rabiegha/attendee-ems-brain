# Workspace — Rabie

Espace de **pilotage personnel** de Rabie. C'est le poste de commande individuel :
ce que je fais **maintenant** et ce que je **capture** au vol.

> Le pilotage est **personnel** (un workspace par dev), mais les dossiers de fond
> (`workstreams/`, `bugs/`, `backlog/`, `debts/`, …) sont **partagés** à la racine de `ems-brain/`.

---

## Fichiers

| Fichier | Rôle | Quand j'y touche |
|---|---|---|
| [NOW.md](NOW.md) | **Piloter** — focus actif + prochaine action + hors-scope | Quand le focus change |
| [INBOX.md](INBOX.md) | **Capturer** — tout en vrac, 1 ligne, taggé | En continu, réflexe |
| [journal/](journal/) | **Historiser** — journal de travail hebdo (1 fichier par semaine ISO) | En fin de journée / semaine |

## La méthode

> **Je capture vite (1 ligne, sans réfléchir dans [INBOX.md](INBOX.md)). Je range plus tard (en mode tri).**

En fin de session (~5 min de tri), je vide l'inbox : chaque ligne part dans le bon dossier partagé
(`../workstreams/`, `../bugs/`, `../backlog/`, `../debts/`, `../rabbit-holes/`, `../learnings/`, `../architecture/`).

Tags d'inbox : `[bug]` `[idea]` `[trap]` `[learn]` `[archi]` `[debt]`.
